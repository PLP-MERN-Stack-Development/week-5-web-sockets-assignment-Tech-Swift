## âš¡ Getting Started

### Prerequisites

- Node.js v18+
- npm
- MongoDB

### Installation

```bash
# Clone the repo
git clone <your-repo-url>
cd week-5-web-sockets-assignment-Tech-Swift

# Server setup
cd server
npm install

# Client setup
cd ../client
npm install
```

### Running the App

```bash
# Start server (in /server)
npm run dev

# Start client (in /client)
npm run dev
```
